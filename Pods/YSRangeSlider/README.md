#### ⚠️ **To use with Swift 2.x please ensure you are using == 2.1.0**
#### ⚠️ **To use with Swift 3.x please ensure you are using >= 3.0.0** 

## How to get started

### install via CocoaPods
```ruby
platform :ios, '8.0'
use_frameworks!
pod 'YSRangeSlider'
```

## Requirements

- Xcode 8.0+
- Swift 3.0+
- iOS 8.0+
